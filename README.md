# PDF to CSV Converter

A simple and efficient tool designed to extract structured data from PDF documents and convert them into clean, organized CSV files. 

## 🚀 Features

* **Accurate Extraction:** Parses text and tabular data from PDF files.
* **Batch Processing:** (Optional) Convert multiple PDFs at once.
* **Clean Output:** Generates structured CSVs ready for data analysis in Excel, Google Sheets, or Pandas.
